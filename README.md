# afterfive

> NYC nightlife planner for interns
> curated events, food, drinks, 
> and workouts on any budget.

afterfivenyc.com

---

## What it is

afterfive helps NYC interns figure out what to do after work without 
spending hours researching. Curated spots, a plan generator, and 
a growing community of interns all on any budget.

## Planned features

- **User accounts** — sign up with work or school email
- **Reviews & ratings** — quick takes on any spot
- **Saved lists** — want to go / been here, shareable with friends
- **Intern-verified badge** — spots with enough positive reviews get marked
- **Social layer** — see where your friends are going

Backend planned with Supabase (auth + database) in a future version.

---

## Tech

- Vanilla HTML, CSS, JavaScript
- Self-hosted on GitHub Pages
- Google Forms for spot submissions
- Spot database managed via spreadsheet
